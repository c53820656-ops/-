Projects & Activities

주요 활동 내용

컴퓨터 구조, 지능형 메모리, 지속 가능한 반도체 기술, 물리 실험 데이터를 중심으로 구재우의 전자공학적 문제 해결력을 정리했습니다.

① Computer Architecture

반도체 아키텍처 심화 탐구

주제: 폰 노이만 병목현상 분석 및 구조적 해결 방안 연구

수행 내용: 기존 폰 노이만 구조의 한계점을 분석하고, 이를 극복하기 위한 하버드 아키텍처 및 GPU의 SIMT(Single Instruction Multiple Threads) 구조를 조사하고 비교했습니다.

성과아키텍처별 데이터 전송 효율성 도표화 및 보고서 작성 역량컴퓨터 구조에 대한 깊이 있는 이해 분석논리적 비교와 구조적 해결 방안 탐색
② Python Simulation

PIM(Processing-In-Memory) 시뮬레이션

주제: Python을 이용한 지능형 메모리 연산 효율성 검증

수행 내용: 메모리 내에서 연산을 수행하는 PIM 방식의 효율성을 확인하기 위해 Python으로 데이터 처리 로직 시뮬레이션을 제작하고 기존 방식과의 속도를 비교했습니다.

성과함수형 로직 기반 시뮬레이터 소스 코드 완성 검증연산 성능 향상 수치 도출 역량알고리즘 설계와 반도체 기술 구현력
③ Sustainable Technology

지속 가능한 기술: 친환경 반도체 연구

주제: 반도체 산업에서의 순환 경제 모델 적용 방안

수행 내용: 반도체 제조 공정의 환경 오염 문제를 분석하고, ESG 관점에서 지속 가능한 제조 공정과 자원 재활용 시스템을 연구했습니다.

성과안산시 상호문화 자본 개념과 기술의 사회적 가치 연계 발표융합 주제 발표 자료 제작 역량사회적 책임감과 융합적 사고력
④ Physics Experiment

물리 실험: 역학적 에너지 보존 법칙

주제: 경사면 카트 실험을 통한 물리 법칙 데이터 분석

수행 내용: 경사면을 이동하는 카트의 질량과 각도를 변수로 하여 위치 에너지와 운동 에너지의 전환 과정을 정밀 측정했습니다.

성과미적분을 활용한 에너지 변화율 계산 분석데이터 오차 원인 분석 보고서 작성 역량정밀 측정과 물리 법칙의 수학적 증명
