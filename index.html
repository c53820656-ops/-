Self-Introduction

구재우는 시스템의 한계를 질문하고 검증하는 예비 엔지니어입니다.

단순히 지식을 학습하는 데서 멈추지 않고, 왜 병목이 발생하는지와 어떻게 구조를 바꾸면 효율을 높일 수 있는지를 탐구해왔습니다. 폰 노이만 구조, GPU 병렬 처리, PIM(Processing-In-Memory)을 비교하며 이론을 Python 시뮬레이션으로 구현하고 검증하는 경험을 쌓았습니다.
